Appearance

Fast-JSON-UI将 JSON 配置转换为 UI 组件

简单、灵活、高效的 Vue 3 UI 库

快速开始
在 GitHub 上查看
🚀

简单易用

通过简单的 JSON 配置，快速构建复杂的用户界面，无需编写大量的 HTML 和 CSS 代码。

🔄

动态数据绑定

支持变量和方法的动态绑定，轻松实现数据驱动的界面。

🧩

自定义组件扩展

轻松扩展自己的自定义组件，满足各种业务需求。

📱

灵活的布局系统

支持行、列、堆叠等多种布局方式，适应各种界面设计。

⛓️

链式属性访问

支持 user.name 或 array.1.name 这样的链式属性访问，处理复杂数据结构。

🔧

现代 Vue 3 支持

同时提供 Options API 和 Composition API（script setup）组件，适应不同开发风格。
