## 如何使用示例

每个示例都包含以下内容：

1. **目标** - 示例要实现的功能
2. **完整代码** - 可以直接复制使用的代码
3. **分步解析** - 详细解释代码的各个部分
4. **关键技术点** - 示例中使用的关键技术和模式
5. **扩展思路** - 如何扩展和定制示例

您可以直接复制示例代码到您的项目中，或者根据需要进行修改和扩展。
